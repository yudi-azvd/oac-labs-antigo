Intel(R) Quartus(R) Prime Design Suite Release 18.1 README.TXT
==================================================
Note: Intel(R) Quartus(R) Prime includes Quartus Prime Standard and 
Quartus Prime Pro editions. 

This readme.txt file accompanies the Pro, Standard, and Lite editions
of the Intel(R) Quartus(R) Prime Design Suite version 18.1.

Although we have made every effort to ensure that this version
of the Intel(R) FPGA software operates correctly, you can encounter
problems that we have not discovered. If you have a question or
problem that is not answered by the information provided in
this readme.txt file, please go to https://www.intel.com/content/www/us/en/products/programmable.html and click Support, 
or submit a service request at premiersupport.intel.com.

This readme.txt file contains the following information:
*  Intel(R) Quartus(R) Prime Design Suite Contents--lists the set of 
   Intel design tools.
*  Technical Documentation--directs you where to find documentation for 
   Intel design tools.
*  System Requirements--lists the system requirements.

Intel(R) Quartus(R) Prime Design Suite Contents
===================================
*  Intel(R) Quartus(R) Prime Design Software (which includes the 
   Intel(R) FPGA IP Library and Intel(R) Nios(R) II Embedded Design Suite)
*  ModelSim (*)- Intel(R) FPGA Edition Simulation Tool
*  ModelSim (*)- Intel(R) FPGA Starter Edition Simulation Tool
*  Intel(R) FPGA SoC Embedded Development Suite (Pro and Standard Editions)
*  DSP Builder for Intel(R) FPGAs
*  Intel(R) FPGA SDK for OpenCL(TM)
                 
You can download the Intel(R) Quartus(R) Prime software, as well as tools such as the
Advanced Link Analyzer or Stand-Alone Programmer, from the 
Intel FPGA and Programmable Devices Download Center on the Intel website: 
https://www.intel.com/content/www/us/en/programmable/downloads/download-center.html

System Requirements
===================
Operating system support
------------------------
*  Windows 7 SP1(64 bit)
*  Windows 8.1 (64 bit)
*  Windows 10 (64 bit)
*  Windows Server 2008 R2 SP1 (64 bit)
*  Windows Server 2012 R2 (64 bit)
*  Red Hat Enterprise Linux 6 (64 bit)
*  Red Hat Enterprise Linux 7 (64 bit)
*  SUSE SLE 12
*  Ubuntu 14.04.5 LTS (64 bit)
*  Ubuntu 16.04.2 LTS (64 bit)

Notes:
    * The Intel(R) Quartus(R) Prime Software (Pro, Standard, & Lite), 
	  Stand-Alone Programmer (Pro & Standard), and DSP Builder (Pro & Standard)
	  are supported under all listed operating systems.
	* The 32-bit Stand-Alone Programmer is only supported 
	  in Intel(R) Quartus(R) Prime Standard Edition.  
	* Red Hat Enterprise Linux 7 only supports KDE Desktop Manager. 
    * Nios II EDS(Standard and Pro) are not supported under 
      Windows Server 2012 R2, Red Hat Enterprise Linux 7,
	  SUSE, or Ubuntu.
    * OpenCL is supported only under Windows 7 SP1, Windows 8.1,
      Windows 10, RedHat Enterprise Linux 6 and 7, and Ubuntu.
    * SoC Embedded Development Suites (Standard and Pro) are only supported 
	  under Windows 7 SP1, Windows 10, Windows Server 2008 R2 SP1, 
	  Red Hat Enterprise Linux 6 (which requires 32-bit library files).
    * ModelSim-Intel FPGA is supported only under Windows 7 SP1, Windows 8.0,
      Windows 10, Windows Server 2008 R2 SP1, Red Hat Enterprise Linux 6,
	  and 7. All require 32-bit library files.
	* Advanced Link Analyzer is supported only under  Windows 7 SP1, Windows 8.1,
      Windows 10, Windows Server 2008 R2 SP1, and Windows Server 2012 R2.
	* HLS Compiler (Pro and Standard) are supported only under Windows 7 SP1,
	  Windows 8.1, Windows 10, and Red Hat Enterprise Linux 6. 

Operating System Support information including an updated list of limitations for all
editions of the Intel(R) Quartus(R) Prime software can be found on the Intel FPGA website:
https://www.intel.com/content/www/us/en/programmable/support/support-resources/download/os-support.html

DSP Builder 
	Intel(R) Quartus(R) Prime Pro Edition supports MathWorks MATLAB releases 
	2015b, 2016a, 2016b, 2017a, and 2017b.
	Intel(R) Quartus(R) Prime Standard Edition supports MathWorks MATLAB release R2013a.
